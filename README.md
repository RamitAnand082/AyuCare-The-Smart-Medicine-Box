AyuCare: The Smart Medicine Box is a clean and connected solution that helps you stay on top of your pills. It combines IoT hardware, web technologies, and real-time communications in a single integrated platform. The software manages morning, noon, and evening pill consumption and enables modifications prescribed by your doctor through the web interface. Utilizing the ESP32 microcontroller, it comes with an LCD screen, buzzer, and push button to ensure instantaneous notifications and convenient usage experience. Medicine schedules are fetched using REST APIs from the Flask server. The application constantly updates the medicine status based on the user's actions. Each dose is marked as Taken, Missed, or Pending based on how the user responds to alerts. Both audio and LCD notifications are initiated when sending alerts to help minimize skipped doses. Statuses are continuously synced in real-time with the server so your doctor gets insights into your progress without requiring you to visit the hospital or clinic.
